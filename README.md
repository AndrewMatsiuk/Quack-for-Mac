# Quack 🦆 — releases

Auto-update feed for [Quack](https://github.com/AndrewMatsiuk/Quack) (private source).

- `appcast.xml` — the Sparkle feed the app polls for updates.
- Builds are attached to each GitHub Release here (`Quack-<version>.zip`).

This repo holds **only** the compiled app and the update feed — no source code.
